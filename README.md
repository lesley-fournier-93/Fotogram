# 📸 Fotogram

A responsive photo-sharing web application inspired by Instagram, built with HTML, CSS, and JavaScript.

## 🚀 Features

* Display a gallery of images in a clean grid layout
* Responsive design for different screen sizes
* Modern UI inspired by social media platforms
* Structured layout with reusable components

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

* `index.html` – Main structure of the app
* `css/` – Styling and layout
* `img/` – Image assets
* `fonts/` – Custom fonts
* `script.js` – Application logic

## 📚 What I Learned

* Building responsive layouts with CSS
* Structuring a frontend project
* Working with image grids and layouts
* Improving UI/UX design

## 🎯 Purpose

This project was created to practice frontend development and recreate a simple social media-style image gallery.
